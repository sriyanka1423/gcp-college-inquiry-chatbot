# College Inquiry Chatbot using Google Cloud Platform

## Project Overview

This project is a cloud-based college inquiry chatbot developed to answer common student questions about a college, such as academic programs, tuition, housing, and campus life. The chatbot is designed to provide a simple interactive experience where a user first enters personal details and then selects or asks from a list of common college-related questions.

The project is intended to demonstrate the use of cloud technologies for building and deploying an interactive chatbot application.

## Objective

The objective of this project is to create an interactive college inquiry chatbot that:

- Collects basic user information
- Presents a list of common student questions
- Responds with answers to those questions
- Displays a conclusion page with both user and creator details
- Is deployed on Google Cloud Platform and accessible through a public URL

## Platform

This chatbot is deployed using **Google Cloud Platform (GCP)**.

Possible GCP services that can support this project include:

- **Google Compute Engine** for hosting the application
- **Google App Engine** for platform-based deployment
- **Dialogflow** for chatbot interaction and intent handling

## Features

### 1. User Information Collection
The chatbot begins by asking the user to enter:

- First Name
- Last Name
- Email Address

### 2. Student Inquiry List
The chatbot provides four example student questions, such as:

- Does the college have a football team?
- Does it offer a Computer Science major?
- What is the in-state tuition?
- Does it provide on-campus housing?

### 3. Query Responses
The chatbot responds to each of the listed questions with appropriate answers based on the selected college information.

### 4. Chatbot Conclusion
At the end of the interaction, the chatbot displays:

- User First Name
- User Last Name
- User Email Address
- Chatbot Creator First Name
- Chatbot Creator Last Name
- Chatbot Creator School Email Address

### 5. Public Access
The chatbot is accessible through a live GCP deployment URL for instructors and TAs to test.

## Project Workflow

1. User opens the chatbot URL
2. User enters personal details
3. Chatbot displays four common college-related questions
4. User selects or asks one of the questions
5. Chatbot provides the relevant response
6. Chatbot concludes by displaying user and creator details

## Example Questions Supported

The chatbot is designed to answer questions such as:

- Does the college have a football team?
- Does it offer a Computer Science major?
- What is the in-state tuition?
- Does it provide on-campus housing?

These questions can be customized based on the college selected for the project.

## Use Case

This chatbot can help prospective students quickly find answers to basic college-related questions without browsing multiple pages or contacting admissions directly.

## Deliverable

The final deliverable for this project is a **public GCP deployment URL** that allows instructors and TAs to:

- Open the chatbot
- Enter user information
- Ask the listed questions
- View the final chatbot conclusion screen
